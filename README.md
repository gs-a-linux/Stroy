# ⚡ Stroy Terminal (Aether OS v1.0 "Shock")

**Stroy** is a next-generation AI-powered terminal assistant designed for **Aether OS**. Unlike standard terminal wrappers, Stroy operates as a System Intelligence Layer, capable of managing memory, diagnosing errors, and executing complex bash logic with automated permission handling.

Built for power users and developers, Stroy translates human intent into precise, high-performance shell commands.

---

## 🚀 Unique Features (The 10 "God-Mode" Tools)

Stroy provides capabilities that no other terminal AI offers:

1.  **🔍 Dynamic Drive-Search:** Automatically locates "New Volume" or other partitions regardless of mount points.
2.  **🧠 Persistent Memory:** Saves project facts to a local JSON vault so the AI never forgets your paths or preferences.
3.  **🛡️ Smart Sudo-Injection:** Handles automated password piping securely using the `sudo -S` protocol.
4.  **📉 RAM-Purge:** One-click memory optimization by clearing system caches.
5.  **🩺 Log-Sense:** Automatically scans `journalctl` to explain why a command failed.
6.  **🎯 Process-Killer:** Targets applications by name logic (pkill) instead of confusing PIDs.
7.  **🔄 Conversation Context:** Remembers the last 5 commands to resolve pronouns like "it" or "that."
8.  **🏗️ Auto-Venv:** Automatically suggests or creates Python virtual environments for your scripts.
9.  **📜 Script-Gen:** Generates and marks full `.sh` or `.py` files as executable in a single turn.
10. **✨ Clean Protocol:** Uses custom success markers `*()(` and processing markers `*#@` for a distraction-free UI.

## 🛠️ Installation

Installing Stroy as a global command is easy. Run the following in your terminal:

```bash
# 1. Clone the repository
git clone [https://github.com/YOUR_GITHUB_USERNAME/stroy.git](https://github.com/YOUR_GITHUB_USERNAME/stroy.git)

# 2. Enter the directory
cd stroy

# 3. Run the installer (requires sudo for global command creation)
bash install.sh
```
⚠️ TRIAL NOTICE: > This is a preview release.it is optimized for testing. Commands may occasionally require manual correction. The installer will prompt you for your Groq API key and configure the system automatically.
